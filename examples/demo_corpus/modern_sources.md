# Modern Sources

MindSearch motivates dynamic search graphs where a planner decomposes a question into atomic
subquestions and search agents gather evidence for each node. STORM motivates multi-perspective
question asking before synthesis.

RAGentA and MASS-RAG motivate attributed answer generation with specialized evidence-processing
roles. Their lesson for a small prototype is to separate retrieval, extraction, reasoning,
verification, and synthesis rather than relying on one large prompt.

Anthropic's multi-agent research-system write-up motivates orchestrator-worker coordination,
end-state evaluation, external memory, and filesystem-style handoffs from subagents to the
coordinator.
